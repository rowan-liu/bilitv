# BiliTV (MVP)

Channel-surf 风格的 Bilibili 播放器（MVP）。

快速开始：

1. 安装依赖：

   npm install

2. 启动服务：

   npm start

3. 打开浏览器并访问：http://localhost:3000

说明：
- 当前 MVP 使用嵌入式 Bilibili 播放器（iframe）。
- 后续可扩展为使用 yt-dlp 缓存或代理直链播放，但可能会涉及平台限制与合规问题。
